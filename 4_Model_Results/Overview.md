# Model Results

This folder contains all outputs from the analysis.

## Folders

### Tables/
APA 7 formatted tables ready for dissertation:
- **Dissertation_Tables.docx** — All main tables
- **Bootstrap_Tables.docx** — Bootstrap inference results

### Figures/
Publication-ready visualizations:
- Descriptive statistics plots
- Model diagrams
- Effect visualizations

### Summary/
- **Key_Findings_Summary.md** — Plain-language summary of main findings
- `Run_Summary/` — Technical logs and model syntax from the analysis run

### Outputs/
Raw model outputs and fit statistics for further analysis.

## Quick Start

1. Open `Summary/Key_Findings_Summary.md` for a plain-language overview
2. See `Tables/Dissertation_Tables.docx` for formatted results
3. Check `Figures/` for visualizations
