# Codebooks

This folder contains documentation for the survey instruments and variables used in this study.

## Files

### Variable Dictionary
- **Variable_Table.xlsx** — Complete data dictionary with all variables, scales, and roles
- **Variable_Table.csv** — Plain-text version

### Survey Codebooks
- **BCSSE_Codebook.xlsx** — Beginning College Survey of Student Engagement (2024)
- **NSSE_Core_Codebook.docx** — National Survey of Student Engagement (2024) - Core items
- **NSSE_MHW_Codebook.docx** — NSSE Mental Health & Wellbeing module

## About the Surveys

**BCSSE** measures incoming students' high school experiences, expectations for college, and background characteristics.

**NSSE** measures first-year students' participation in educationally effective practices and their perceptions of the college environment.

## How to Use

1. Start with **Variable_Table.xlsx** to see all variables used in the analysis
2. Cross-reference variable names with the survey codebooks for:
   - Original survey question wording
   - Response scale options
   - Variable coding schemes
